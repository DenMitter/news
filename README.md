
## Покрокова інструкція з запуску

1. **Клонуйте репозиторій**

   Спочатку клонувати репозиторій на ваш локальний комп'ютер:

   ```bash
   git clone https://github.com/DenMitter/news.git
   cd news

2. **Налаштуйте Docker**

    Проект використовує Docker для налаштування контейнерів. Щоб почати працювати з проектом, потрібно мати встановлений Docker та Docker Compose. Завантажте їх з офіційного сайту:

    <a href="https://www.docker.com/products/docker-desktop/">Docker</a>

    <a href="https://docs.docker.com/compose/install/">Docker Compose</a>


3. **Запустіть проект за допомогою Docker**

    У каталозі проекту створіть Docker-контейнер та запустіть його за допомогою Docker Compose:

       docker-compose up --build
   
    Це запустить контейнер з необхідними службами, які забезпечують запуск веб-додатка.


4. **Перевірте роботу проекту**

    Після успішного запуску, ви зможете переглянути проект у браузері за адресою:

        http://localhost:8080


5. **Налаштування Nginx**

    У проекті є конфігураційний файл Nginx для налаштування веб-сервера. Цей файл знаходиться в каталозі nginx/. Ви можете редагувати його відповідно до своїх потреб.
